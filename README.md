## Installation

ZHKQRCode is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'ZHKQRCode', :git => 'https://github.com/ZHK1024/ZHKQRCode.git', :tag => version
```
